# Changelog

All notable changes to TaskTrack Pomodoro will be documented in this file. This project adheres to a manual release process; update both this file and `assets/changelog.json` when shipping new versions so the in-app update summary stays accurate.

## [0.0.1] - 2025-12-07
- Add daily task planner with active task selection and pomodoro counts.
- Introduce focus/short/long timer with alarm and task-linked completion tracking.
- Show today summary and last-7-days pomodoro totals, stored per date.
- Replace branding with a stopwatch-inspired logo and update manifest text.
- Namespace localStorage keys to avoid collisions with other template-based apps.

## [0.0.0] - 2025-10-11
- Create base Progressive Web app.
